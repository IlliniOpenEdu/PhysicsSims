# PhysicsSims

This is a simple physics simulation project that demonstrates basic principles of motion, forces, and collisions. The project is fully created in the frontend using TypeScript and React, and it allows users to run the engine within the browser.

## Get Started

To get started with the PhysicsSims project, follow these steps:

1. Clone the repository:
```bash
git clone https://github.com/Edoubek1024/PhysicsSims.git
```
2. Navigate to the project directory and download the dependencies:
```bash
cd PhysicsSims
npm install
```
3. Start the development server:
```bash
npm run dev
```
